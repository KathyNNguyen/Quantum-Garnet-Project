Meet the Team:

Shanie Portal

I contributed to the project by setting up Flask, organizing the project structure, and writing the HTML and CSS for all pages. I implemented button functionalities and page redirects to streamline user navigation.

On the backend, I created and managed the user_account and promotions databases. I developed the account creation, login, and logout functionalities, integrating the user_account database to validate credentials. I also built the account feature to display user information upon login and the directory database display.

I developed the promotions feature, which dynamically shows promotions based on user tier, and implemented the flipbook display on the About page to highlight our team's' contributions. Kathy Nguyen

I contributed to mostly frontend components of the project with the Slot Maps and the QC rewards page. Non-frontend contributions include connecting the backend to frontend with Javascript, and creating the README page for viewers to download dependencies and view the application.

The Slot Maps page contains the floor plan of the casino and coordinates for over 100+ machines, pulled from the slot_machines database. To filter out through the various slot machines, there is a search functionality for users to look for their favorite game.

The QC Rewards page is still in progress and will be updated here.

Brynn Loeffler

I contributed to the project by setting up our search bar, and getting it to display our slot machines and allow the user to sort by the name of the slot machine. I also recorded the majority of team meetings and compiled them into a playlist.

The search bar is completely hooked up to the slot_machine database, and hides any machines that don't contain whatever is searched within the name of the machine. It also displays the min and max bets for each machine to the user for clarity.

The team meeting playlist contains the majority of the team meetings, allowing us to track goals and progress in the project. It also allows us to track participation within the group by who shows up to the meetings.

Elan Golan

This data base stores all of the data that we used for the games. This simulated the real data that would be pulled on a nightly basis that would give us the answers we are looking for in optimizing slot machine placement.
